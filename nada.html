Elemen HTML

  • <h1> - <h6>: Digunakan untuk judul dengan tingkat penting.
  • <p>: Digunakan untuk paragraf.
  • <a>: Digunakan untuk tautan.
  • <img>: Digunakan untuk gambar.
  • <html>: Digunakan untuk menandai seluruh dokumen HTML.
  • <head>: Digunakan untuk bagian metadata dari dokumen HTML, seperti judul dan link ke file eksternal.
  • <title>: Digunakan untuk menetapkan judul dokumen yang tampil di tab browser.
  • <body>: Bagian utama dokumen yang berisi konten yang ditampilkan di browser.
  • <h1> hingga <h6>: Digunakan untuk membuat judul dengan tingkat kepentingan yang berbeda, dari terbesar (<h1>) hingga terkecil (<h6>).
  • <p>: Digunakan untuk membuat paragraf teks.
  • <a>: Digunakan untuk membuat hyperlink yang menghubungkan ke halaman lain atau sumber daya lainnya.
  • <img>: Digunakan untuk menampilkan gambar dalam halaman web.
  • <ul>: Digunakan untuk membuat daftar tidak terurut (dengan bullet).
  • <ol>: Digunakan untuk membuat daftar terurut (dengan nomor).
  • <li>: Digunakan untuk membuat item dalam daftar (baik di dalam <ul> atau <ol>).
  • <div>: Digunakan untuk membungkus dan mengelompokkan elemen HTML lainnya secara blok.
  • <span>: Digunakan untuk membungkus bagian teks atau elemen inline lainnya.
  • <form>: Digunakan untuk membuat formulir interaktif untuk input data pengguna.
  • <input>: Digunakan untuk elemen input dalam formulir (seperti kotak teks, tombol, atau checkbox).
  • <button>: Digunakan untuk membuat tombol yang dapat diklik oleh pengguna.
  • <table>: Digunakan untuk membuat tabel di halaman web.
  • <tr>: Digunakan untuk mendefinisikan baris dalam tabel.
  • <td>: Digunakan untuk mendefinisikan sel data dalam tabel.
  • <th>: Digunakan untuk mendefinisikan sel header dalam tabel.
  • <label>: Digunakan untuk memberi label pada elemen input dalam formulir.
  • <link>: Digunakan untuk menghubungkan dokumen HTML ke file eksternal, seperti file CSS.
  • <meta>: Digunakan untuk menyimpan metadata tentang halaman, seperti charset atau deskripsi.
  • <strong>: Digunakan untuk memberi penekanan pada teks, umumnya tampil dalam bentuk tebal.
  • <em>: Digunakan untuk memberi penekanan pada teks, umumnya tampil dalam bentuk miring.
  • <br>: Digunakan untuk membuat pemisah baris (line break) dalam teks.
  • <hr>: Digunakan untuk membuat garis horizontal sebagai pembatas atau pemisah.
  • <audio>: Digunakan untuk menyematkan elemen audio dalam halaman web.
  • <video>: Digunakan untuk menyematkan elemen video dalam halaman web.
  • <iframe>: Digunakan untuk menampilkan halaman web lain di dalam halaman HTML yang sedang aktif.
